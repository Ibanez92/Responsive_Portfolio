Recipe App

https://ibanez92.github.io/Recipe-App/

Recipe app is a static mobile/desktop responsive web application that uses Javascript to dynamically interact with the Edamame Recipe Search API.The user inputs any type of food and API brings up the name of the Recipe, the Calories, the Diet Label, and the Health Label. Clicking on the view recipe button will take the user to the source website to find the ingredients of the recipe chosen.

Movie-App

https://ibanez92.github.io/Movie-App/

Movie App is a static desktop application that uses vanilla Javascript/ES6 and TMDB (The Movie Database) API. When the user inputs the movie title into the search bar, the user's input title will show movie images and movie trailers from youtube when the movie image is clicked. Then in order from top to bottom, it shows a row of the upcoming movies, the popular movies, and the top-rated movies. All 4 are called from the TMDB (The Movie Database) API database.
