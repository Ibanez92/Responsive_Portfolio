Recent Projects

U8 Spartan Soccer

u8-spartan.onrender.com

Leveraging Flask, a Python micro web framework, as the backend foundation, the site employs HTML, CSS, and JavaScript to structure, style, and enhance its web pages. The responsive design is achieved through Bootstrap, ensuring the website's adaptability across various devices. The interactive calendar is powered by the FullCalendar JavaScript library, enabling users to stay updated on team events. Integrating the Google Sheets API facilitates real-time data retrieval and display from a Google Sheets spreadsheet. The Jinja2 templating engine dynamically renders content, enhancing the site's flexibility. Finally, hosting on the Render platform ensures reliable and efficient deployment.

TicTacToe

https://tictactoe-e327a.web.app/

I have created an engaging TicTacToe application using the Angular framework in JavaScript. Leveraging the power of Angular, I have built a dynamic and interactive gaming experience. The application is deployed as a Progressive Web Application (PWA) on the Firebase platform, providing seamless accessibility across devices. Recipe App

https://ibanez92.github.io/Recipe-App/

The Recipe app is a responsive web application that utilizes JavaScript to interact with the Edamame Recipe Search API. Users can search for recipes by entering any food type, and the API fetches the recipe name, calories, diet label, and health label. The app includes a "View Recipe" button that directs users to the source website for detailed ingredients and instructions. It provides a seamless and convenient way to explore and discover a variety of delicious recipes.
